What did I do today?

Today was the CSS challenge on codepen for this week.
I started with cloning, then worked on a workshop about :has().
I practiced adding features like changing borders, showing a background photo on click, adding button shadows, and moving elements to different areas.
Examples:
Change background color for a list item with an h2:
li:nth-of-type(1):has(h2) {
background-color: var(--goud); }
Load a background image if a checkbox is checked:
li:nth-of-type(3):has(:checked) {
background-image: url("https://assets.codepen.io/274456/photo-grijs.png"); background-size: cover; }
I worked on versions 1 and 2 of the CSS challenge and plan to improve them further to learn more.

What did I learn today?

I learned how to use the :has() selector effectively, how to style elements dynamically, and how to work with shadows, background images, and interactive elements in CSS.
This helped me understand how to create responsive and dynamic designs.
